# Скрипты запуска и тестирования

## Запуск приложений

### API сервер
```bash
./run_api.sh
```
Запускает REST API сервер для транскрибации аудио файлов.

### CLI интерфейс
```bash
./run_cli.sh
```
Запускает командную строку для транскрибации файлов.

### GUI интерфейс
```bash
./run_gui.sh
```
Запускает графический интерфейс пользователя.

**Windows:** используйте `run_gui.bat` — он активирует окружение conda `gigaam` и запускает `app.py`. Чтобы создать ярлык на рабочем столе, выполните один раз:
```powershell
powershell -ExecutionPolicy Bypass -File scripts\create_desktop_shortcut.ps1
```

## Скрипты тестирования

### Проверка настройки API
```bash
python3 test_api_setup.py
```
Проверяет корректность настройки API, переменных окружения и доступность эндпоинтов.

### Тест множественной загрузки
```bash
python3 test_batch_download.py
```
Тестирует логику создания ZIP архивов для скачивания нескольких результатов.

### Тест исправления транскрибации
```bash
python3 test_transcription_fix.py
```
Тестирует исправление чтения результатов транскрибации из файлов.

## Примечания

- Все скрипты запуска используют conda окружение `gigaam`
- Тестовые скрипты написаны на чистом Python без внешних зависимостей
- Скрипты совместимы с Linux/macOS; для Windows предусмотрены `run_gui.bat` и `create_desktop_shortcut.ps1`
